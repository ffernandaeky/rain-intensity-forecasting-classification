# Project Magang: Forecasting dan Klasifikasi Intensitas Hujan

Proyek ini bertujuan untuk menganalisis serta memprediksi intensitas curah hujan menggunakan pendekatan machine learning dan time series forecasting. Fokus utamanya adalah membantu memahami pola hujan dari data historis, lalu mengklasifikasikan kondisi hujan menjadi kategori yang lebih informatif.

## Apa yang dilakukan dalam proyek ini?

- Menganalisis data curah hujan dan stasiun BMKG
- Melakukan pemodelan forecasting untuk memprediksi curah hujan ke depan
- Menggunakan metode klasifikasi untuk mengidentifikasi intensitas hujan
- Membandingkan beberapa pendekatan model, seperti ARIMA, Exponential Smoothing, dan PyCaret

## Tujuan proyek

Proyek ini dibuat untuk memberikan wawasan yang lebih baik mengenai pola cuaca, khususnya curah hujan, sehingga hasil analisis dapat dimanfaatkan untuk keperluan prediksi dan pengambilan keputusan yang lebih efektif.

## Teknologi yang digunakan

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- PyCaret
- Statsmodels
- Matplotlib / Plotly

## Struktur utama proyek

- Notebook forecasting dan klasifikasi: berisi proses analisis, pemodelan, dan evaluasi
- Data mentah: digunakan sebagai sumber input untuk pelatihan dan prediksi
- Hasil model: menyimpan output forecasting dan evaluasi performa model

## Catatan penting

Beberapa file hasil pelatihan, artifact model, dan data output besar sengaja tidak dimasukkan ke repository agar proyek tetap rapi dan lebih efisien untuk dikelola.

## Penutup

Proyek ini merupakan bagian dari perjalanan magang yang menggabungkan analisis data, pemodelan prediktif, dan penerapan machine learning untuk memahami fenomena cuaca secara lebih sistematis.
